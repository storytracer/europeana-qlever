
# Media Formats/Mime Types

The tables below list the mime types of media resources included in Europeanaā€™s manifests. We have categorised these mime types into two groups, based on how they are represented in the canvas:

- [(Browser) Supported](#browser-supported)
- [Specialised](#specialised)

Mime types not listed on this page are not supported by Europeana, meaning they are not included in the manifest and consequently not displayed.
