# EDM profiles

As Europeana develops new data features, it continually refines and extends the Europeana Data Model (EDM) to meet evolving requirements. These EDM profiles are specifically designed to address the needs of both data partners and the core Europeana platform. While the Europeana Foundation coordinates and maintains these profiles, their development is a collaborative process, stemming from community-led working groups and task forces, as well as internal technical requirements.
