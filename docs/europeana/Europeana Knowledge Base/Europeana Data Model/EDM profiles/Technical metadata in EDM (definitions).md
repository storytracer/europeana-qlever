# Technical metadata in EDM (definitions)

To support the application of the [Media policy](../../Media%20policy.md) and provide additional useful facets and filtering of Europeana resources, the addition of technical metadata at the level of the `edm;WebResource`, which was currently absent from EDM, is needed. This specification applies to five media types which Europeana currently supports, namely: Sound, Video, Text, Image and 3D objects. This profile lists the properties that will apply to the `edm:WebResource` class and an additional class that were defined to support such functionality.

# Approach

A minimal approach has been adopted to the addition of properties to EDM. Not all functions require the addition of a specific property. Consider for example the filtering of Europeana objects based on the presence of a usable thumbnail. This can be detected by checking for the presence of a URL in the “edm:preview” property (and the resource flagged accordingly in a search system) without the need for a specific EDM property to hold that flag in Europeana’s reference data.

Some values can be used to generate further values and, depending on the stage in the processing of data where this happens, a property may be needed to record the output. For example, the orientation of an image will be generated by processing numeric values given in the height and width properties, giving the result as “Portrait” or “Landscape”. A few properties have been created in the EDM namespace even though similar properties already exist elsewhere (e.g. [Exif](https://www.w3.org/2003/12/exif/) and [EBUCore](https://tech-metadata.ebu-it-tools.ch/ontologies/ebucore/index.html)). This is because the allowable values did not match our requirements at this time (e.g. EDM needs literal values in some cases).  
These properties are: “[edm:hasColorSpace](https://europeana.atlassian.net/wiki/spaces/EF/pages/edit-v2/2806513675#edm%3AhasColorSpace)”, “[edm:componentColor](https://europeana.atlassian.net/wiki/spaces/EF/pages/edit-v2/2806513675#edm%3AcomponentColor)”, “[edm:spatialResolution](https://europeana.atlassian.net/wiki/spaces/EF/pages/edit-v2/2806513675#edm%3AspatialResolution)” and “[edm:codecName](https://europeana.atlassian.net/wiki/spaces/EF/pages/edit-v2/2806513675#edm%3AcodecName)”.

All the properties defined below should be applied to[edm:WebResource](../../EDM%20-%20Mapping%20guidelines/EDM%20Core%20classes/edm_WebResource.md).

# Properties

## Property edm:codecName

|                             |                                                                                                                                       |
|:----------------------------|:--------------------------------------------------------------------------------------------------------------------------------------|
| **URI**                     | `http://www.europeana.eu/schemas/edm/codecName`                                                                                       |
| **Label**                   | Codec                                                                                                                                 |
| **Definition**              | The name of a device or computer program capable of encoding or decoding a digital data stream or signal (i.e. **co**der-**dec**oder) |
| **Domain**                  | `edm:WebResource`                                                                                                                     |
| **Obligation & Occurrence** | Optional (Minimum: 0, Maximum: 1)                                                                                                     |
| **Example**                 | mp4v                                                                                                                                  |
| **Comment**                 | Used for video                                                                                                                        |

## Property ebucore:hasMimeType

|                             |                                                                                                                                                                                              |
|:----------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **URI**                     | `https://www.ebu.ch/metadata/ontologies/ebucore/ebucore#hasMimeType`                                                                                                                         |
| **Label**                   | MIME Type                                                                                                                                                                                    |
| **Definition**              | The main MIME types as defined by IANA: e.g. audio, video, text, application, model, or a container MIME type.                                                                               |
| **Usage note**              | In oEmbed web resources, when the URI of the web resources includes the format parameter, the value of `ebucore:hasMimeType` must contain the MIME that corresponds to the format parameter. |
| **Subproperty of**          | `ebucore:hasFormat`                                                                                                                                                                          |
| **Domain**                  | `edm:WebResource`                                                                                                                                                                            |
| **Range**                   | `rdfs:Literal` with controlled values taken from the [IANA list of internet media types](https://www.iana.org/assignments/media-types/media-types.xhtml).                                    |
| **Obligation & Occurrence** | Optional (Minimum: 0, Maximum: 1)                                                                                                                                                            |
| **Example**                 | image/jpeg                                                                                                                                                                                   |

# Classes

## Class edm:FullTextResource

|                 |                                                                                                                                                                                                                                                       |
|:----------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **URI**         | `http://www.europeana.eu/schemas/edm/FullTextResource`                                                                                                                                                                                                |
| **Label**       | Full-text Resource                                                                                                                                                                                                                                    |
| **Definition**  | A resource that has machine readable full-text content.                                                                                                                                                                                               |
| **Subclass of** | `edm:WebResource`                                                                                                                                                                                                                                     |
| **Example**     | Resources containing text resulting from Optical Character Recognition (OCR).                                                                                                                                                                         |
| **Comment**     | When full-text is available for a digital object it must be represented in this specific class because in some cases the `edm:WebResource` that provides the view of the object to end-users does not contain the full-text in machine readable form. |
