# EDM profile for Sound (definitions)

The EDM profile for Sound has emerged from task force discussions that took place between **1 March 2014 to 30 September 2014** as part of the Europeana Sounds project. This task force consisted of Europeana Sounds project members as well as representatives of the wider EuropeanaTech community. The final report on EDM profile for Sound was delivered as part of the task 1.3 EDM profile and it describes a set of recommendations for the adaptation of the current Europeana Data Model (EDM) for audio and audio-related objects. The profile suggests many new properties, including several properties for technical metadata already defined in the [Technical metadata in EDM (definitions)](Technical%20metadata%20in%20EDM%20(definitions).md). An overview of all new classes and properties is available [here](https://docs.google.com/spreadsheets/d/1hOYzRA6igftJZZaRLQN0g-y3SmZV2utfvYpb7I6DBPo/edit?gid=0#gid=0).
