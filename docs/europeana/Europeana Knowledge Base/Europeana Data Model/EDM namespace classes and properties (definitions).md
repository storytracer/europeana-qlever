# EDM namespace classes and properties (definitions)

# Introduction

This page provides the authoritative definitions of all classes and properties in the EDM namespace. It covers elements forming the core EDM used by data providers, elements used internally by Europeana, and all elements defined within the EDM profiles.

## Class edm:ProvidedCHO

|                |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|:---------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **URI**        | http://www.europeana.eu/schemas/edm/ProvidedCHO                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Label**      | Provided Cultural Heritage Object                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Definition** | Represents the cultural heritage objects that Europeana collects descriptions about.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Notes**      | EDM enables the integration of data from multiple providers describing the same real-world resource, while maintaining explicit provenance for each data source. EDM extends the Object Re-use and Exchange (ORE) model by adopting its [proxy mechanism](https://www.openarchives.org/ore/1.0/datamodel#Proxy), which allows distinct provider-specific descriptions to coexist without loss of attribution. In this architecture, `edm:ProvidedCHO` functions as the central aggregation node, serving as the reference point for the resource and mediating its multiple contextual instantiations via proxies. The use of `edm:ProvidedCHO` differs between data providers and aggregators (including Europeana). Data providers typically attach descriptive statements about the cultural heritage object (CHO) directly to the `edm:ProvidedCHO`. Aggregators, by contrast, rely primarily on the proxy mechanism to ensure that attribution to the original data source is preserved. In the internal representation of EDM at Europeana (EDM Internal), the `edm:ProvidedCHO` is restricted to a single property—`owl:sameAs`. In the external representation (EDM External), however, all properties applicable to `ore:Proxy` may also be asserted on `edm:ProvidedCHO`. |
| **See also**   | See the usage of [edm:ProvidedCHO in EDM external and EDM internal.](https://europeana.atlassian.net/wiki/spaces/EF/pages/edit-v2/3138879489#edm%3AProvidedCHO)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
