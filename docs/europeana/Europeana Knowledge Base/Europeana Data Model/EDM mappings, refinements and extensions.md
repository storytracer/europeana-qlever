
# EDM mappings, refinements and extensions

Europeana has collected mappings to EDM, refinements and extensions created by data providers while submitting data to Europeana. If you are looking for a specific mapping to EDM or for inspiration for making a system that uses EDM interoperable with other data models, you can see them on the [page of the EuropeanaTech Task force that started this effort](https://pro.europeana.eu/project/edm-mappings-refinements-and-extensions). You can also submit new ones - please contact us!
