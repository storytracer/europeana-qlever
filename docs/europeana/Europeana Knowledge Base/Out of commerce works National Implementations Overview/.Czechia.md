| Scope of exception               | No subject-matter of protection is excluded. Reproduction and communication to the public, including making available to the public, on non-commercial websites, for non-commercial purposes.                                                                                                                                                                                                                                                                                                                                                                                       |    |    |    |    |    |    |

| Other conditions                 | The list of out of commerce works under the system existing since 2017 before the adoption of the Copyright in the Digital Single Market Directive is continuously checked against available databases of the booksellers/distributors, as well as against the National Bibliography database.                                                                                                                                                                                                                                                                                      |    |    |    |    |    |    |
