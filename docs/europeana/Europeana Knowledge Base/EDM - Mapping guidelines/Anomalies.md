# Anomalies

- Definition of edm:hasType in the EDM specification implies that it is for “What” questions in the portal by acting as an umbrella super-property for dc:type and format (and possibly others). It therefore has the domain of Europeana CHOs and range of Concepts. It has however also been included in the edm:Event class to allow the typing of Event entities
